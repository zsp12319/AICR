# AICR
我构建了一个基于 Mimo 大模型的多 Agent 智能代码审查与自动修复系统（AICR）。它由静态分析 Agent、语义推理 Agent 与修复测试 Agent 三方协作，实现从 PR 提交到自动修复、验证的全闭环。  核心痛点：团队 Code Review 耗时巨大，且人类审查容易遗漏并发安全、边界条件等语义层面的隐蔽缺陷，导致技术债持续堆积。  核心逻辑流：  静态分析 Agent 对 PR 代码执行 lint/SAST 扫描，提取可疑片段并结构化输出；  语义推理 Agent 采用长链思维，结合 diff 上下文、调用链及业务逻辑，进行多步因果推理，精准判定真实缺陷并生成修复建议；  修复测试 Agent 负责应用补丁、自动生成并运行单元/集成测试，若失败则触发反思-重试回路
